# F.A.R.T.
### Flatulence Activated Response Technology

---
## How It Works / 動作概要

This mod adds beautifully immersive fart technology
to various player actions.

Fully multiplayer synchronized,
and yes — enemies can hear everything.

**Huntsmen can and will investigate suspicious ass activity.**

プレイヤーの各種アクションに応じて、
音色豊かな放屁機能を提供します。

マルチプレイ完全対応で、
もちろん敵にだって聞こえています。

**ハンターはあなたの放屁を確実に仕留めます。**

---

## FEATURES / 主な機能

- Adds fart events to multiple player actions
- Individual fart chance settings per action
- Huntsmen investigate suspicious fart activity
- Multiplayer compatible
- Randomized pitch variation
- Custom MP3 replacement support (local playback only)

- プレイヤーの各種アクション時に放屁を追加
- アクションごとに発生率を設定可能
- ハンターが放屁を入念にチェックしています
- **マルチプレイ対応**
- ランダム音程変化機能付き
- 放屁音MP3の差し替えに対応（ローカル再生のみ）

## SUPPORTED ACTIONS / 対応アクション

- Tumble
- Jump
- Sliding
- Sprint
- Crouch

Each action supports:
- Enable / Disable
- Individual fart chance settings (0%–100%)
Fart trigger rates are based on each player's own config.

So if you wish to become
a permanent biological warfare unit,
the choice is yours.
**(We are not responsible for lost friendships.)**

- 転倒
- ジャンプ
- スライディング
- ダッシュ
- しゃがみ

それぞれに有効/無効、
抽選確率（0%~100%）を設定可能です。
また、放屁抽選率は各プレイヤーの設定値を使用します。
俺は放屁マンになりたい！という者は望むままに。
**（友人を失っても責任は負いかねます）**

## CONFIG / 設定

All settings can be configured through the in-game MODS config menu.
各種設定はゲーム内の「MODS」コンフィグから変更可能です。

- Volume (0% - 100%)
Fart volume
放屁音量

- Enabled
Enable / Disable per action
アクション別の有効/無効

- FartRate (0% - 100%)
Fart trigger chance per action
アクション別の放屁発生率

- RandomPitchEnabled
Enable random pitch variation
音程変化機能の有効/無効

- PitchVariation (0% - 100%)
Pitch variation range
音程変化幅

## MULTIPLAYER / マルチプレイ

This mod is fully multiplayer compatible.
All fart events are synchronized through REPOLib.
*(Yes. These are actual fart packets.)*

For best audio consistency,
installation on both host and client is recommended.

このMODはマルチプレイ対応です。
すべての放屁イベントは同期的に管理されます。
*（ワオ、これがおならパケットね）*

音声同期の関係上、
ホスト・クライアント双方への導入を推奨します。

## CUSTOM SOUNDS / 音声差し替え

Simply place your own .mp3 files next to the DLL
to use custom fart sounds.

Note:
Audio files themselves are played locally only.

DLLと同じ場所に mp3 を置くだけで、
放屁音を自由に差し替え可能です。
※ 音声ファイルはローカル再生です。

### Examples / 配置例

```txt
plugins/F.A.R.T/
    F.A.R.T.dll
    ultra_fart.mp3
    wet_one.mp3
    tactical_assault.mp3
```
